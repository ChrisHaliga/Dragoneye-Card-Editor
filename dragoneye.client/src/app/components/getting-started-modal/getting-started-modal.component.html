Getting Started with Dragoneye Card Editor
Welcome to Dragoneye!

Create, edit, and manage your card collections with ease. Here's how to get started:

Creating Your First Card
  • Use the Project Manager panel on the left to organize your cards into groups
  • Click the + Add Card button to create a new card
  • Select your new card to start editing its properties
Editing Card Properties
  • Use the Card Properties panel at the bottom to edit card details
  • Set the card's title, type, element, and background image
  • Add multiple card details with action points (AP) and spell points (SP) costs
  • Each detail can have its own description and costs
Managing Your Collection
  • File → Save to save your card set to the server
  • File → Load to browse and load existing card sets
  • File → Export as JSON to download your cards as a backup
  • File → Export as SVG to export individual cards as images
Interface Tips
  • Use View menu to show/hide panels based on your workflow
  • The card preview updates in real-time as you make changes
  • Use zoom controls in the card display to get a better view
  • Elements have unique symbols and colors for easy identification
Pro Tip: Start with simple cards and gradually add more complex details as you become familiar with the interface!
