Getting Started with Dragoneye
Welcome to Dragoneye Card Editor!

Create, edit, and manage your custom trading cards with a professional interface. This guide will help you get started with the key features.

Creating Your First Cards
  • Add a Group: Click the "+ Group" button in the Card Hierarchy panel
  • Add Cards: Click "+ Add Card" within any group to create new cards
  • Edit Properties: Select a card and use the Card Properties panel at the bottom
  • Add Details: Click "+ Add Detail" to add actions, spells, or abilities to your cards
Managing Your Card Sets
  • Save: Use Ctrl+S or File → Save to save your current work
  • Save As: Use Ctrl+Shift+S or File → Save As to create a new file
  • Load: Use Ctrl+O or File → Load to open existing card sets
  • Export: Export your cards as JSON or individual cards as SVG images
  • Import: Import card sets from JSON files
Interface Overview
  • Card Hierarchy: Left panel for organizing groups and cards
  • Card Display: Center area showing your card design with zoom controls
  • Card Properties: Bottom panel for editing selected card details
  • Collapsible Panels: Click the arrow buttons to hide/show panels
Keyboard Shortcuts
  • Ctrl+S: Quick save to current file
  • Ctrl+Shift+S: Save As (create new file)
  • Ctrl+O: Open/Load file dialog
  • Ctrl+E: Export current set as JSON
  • Ctrl+,: Open preferences
  • Esc: Close modals and deselect details
Customization
  • Themes: Switch between Light, Dark, or Auto (system) themes
  • Auto-Save: Configure automatic saving at custom intervals
  • Default Settings: Set your preferred element and card type for new cards
  • Elements: Choose from Fire, Water, Earth, Air, Dark, Light, and Arcane
Pro Tips
  • Use the zoom controls in the top-right of the card display to get a better view
  • Click and drag to pan around zoomed-in cards
  • Group names can be edited by clicking the pencil icon next to them
  • Card details can be collapsed/expanded by clicking on them
  • Use the dropdown menus on cards and groups for quick actions like duplicate and delete
